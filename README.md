# umi project

## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```

### 创建项目  yarn create @umijs/umi-app

+ 创建项目  yarn create @umijs/umi-app

### 修改配置文件

+ .umirc.ts文件是默认的配置文件，包括内置功能和插件功能配置
+ 删除.umirc.ts文件，新建config文件夹，文件见中新建config.ts文件，该文件用于项目配置
+ config下创建proxy.ts，用于请求代理
